# NHA Hackathon: PS1 Claim Classification

This project contains code and data artifacts used for a National Health Authority (NHA) hackathon submission focused on automated claim classification and processing. The repository includes sample claim sets organized by region and facility under the `Claims/` directory and interactive analysis in `main.ipynb`.

## Repository Structure

- `main.ipynb` — analysis and modeling notebook
- `nha_client.pyw` — helper scripts / client utilities
- `Claims/` — organized claim datasets used for training and evaluation

## How to run

1. Open and run `main.ipynb` to reproduce preprocessing, training, and evaluation steps.
2. Use `nha_client.pyw` for any dataset utilities or quick local tooling.

## Results

Final evaluation: F1 score = 0.72
